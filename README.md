# tugas_HTML1

1. html - pembungkus utama seluruh dokumen HTML. Semua elemen yang ada di dalam halaman web harus berada di dalam tag ini agar bisa dibaca dengan benar oleh browser.
2. head - berisi metadata atau informasi tentang halaman, seperti judul, link CSS, dan pengaturan lainnya yang tidak ditampilkan langsung di halaman website.
3. title - digunakan untuk menentukan judul halaman yang muncul di tab browser, sehingga memudahkan pengguna mengenali halaman yang sedang dibuka.
4. body - berisi semua konten utama yang akan ditampilkan di halaman web, seperti teks, gambar, link, dan elemen lainnya.
5. h1 - digunakan untuk membuat heading atau judul utama pada halaman. Tag ini memiliki ukuran paling besar dan biasanya digunakan sebagai judul utama sebuah halaman.
6. p - digunakan untuk membuat paragraf teks agar isi tulisan lebih rapi dan mudah dibaca oleh pengguna.
7. a - digunakan untuk membuat hyperlink atau tautan yang bisa mengarahkan pengguna ke halaman lain, baik di dalam website yang sama maupun ke website lain.
8. img - digunakan untuk menampilkan gambar pada halaman website dengan menggunakan atribut `src` sebagai sumber gambar dan `alt` sebagai deskripsi gambar.
9. ul & li - digunakan untuk membuat daftar tidak berurutan (bullet list), di mana `<ul>` sebagai wadah daftar dan `<li>` sebagai item di dalam daftar tersebut.
10. div - digunakan sebagai container atau wadah untuk mengelompokkan elemen HTML agar lebih mudah diatur, baik untuk keperluan desain (CSS) maupun pengelolaan struktur halaman.
